## 2. 레지스터: CPU 내 저장 공간
- EIP(Extend Instruction Pointer): PC(Program Counter)와 비슷함.
- ESP(Extend Stack Pointer): 스택의 시작이 어디까지 인지 알려주는 레지스터
- EBP(Extend Base Pointer): 스택의 기준점, 한수 호출 시만 바뀜.
- FLAGS: 계산 결과에 따른 값을 저장
### 연산에 쓰이는 레지스터
- EAX
- EBX
- ECX
- EDX
- ESI
- EDI


## 3. 기초 어셈블리 명령어
- push: 스택에 값을 넣기
- pop: 스택에서 값을 빼기
- add: (+) 연산
- sub: (-) 연산
- mov: 대입연산
- xor: xor 연산 [예: xor 8, 8 => 0]
- add, sub, move, xor 앞에 위치한 레지스터에 결과 저장(window), 리눅스인 경우 뒤에
***
- jmp: 해당 주소로 이동, 이 명령은 해당 주소를 EIP에 대입
- cmp: 비교 FLAG에 같으면 0 다르면 1 대입
- je: jump equal
- jne: jump not equal
- call: 함수 호출 시 사용. push EIP(-> 리턴 주소)를 먼저 수행 후 이동

## 6. 조건 분기문
조건 분기문의 경우 다음과 같은 순서로 작동합니다:
1. cmp나 test명령으로 두 피연산자를 비교
2. jump 명령어를 이용해 해당 주소로 이동 (조건에 따름) 

참고로, jump 계열 명령어는 다음과 같습니다:
- e = equal
- g = greater than
- l = less than
- n = not
